//	УСТАНОВКА
// ==================================================
1. Установить node.js http://nodejs.org/

2. В папке builder открыть командную строку и выполнить комманды:
	npm install gulp -g
	npm update
	gulp


//	ЗАПУСК ПРОЕКТА
// ==================================================
Открыть командную строку и выполнить:
	gulp	
Или открыть файл gulp-run.cmd

Запустится сервер по адресу http://localhost:8080
Сервер автоматически обновляет страницу в браузере при изменении файлов с версткой (Stylus, Jade, JavaScript).
Подробнее в gulpfile.js



//	СТРУКТУРА ПРОЕКТА
// ==================================================
gulpfile.js ....... Настройки Gulp
gulp-run.cmd ...... Запуск Gulp
site/ ............. Скомпилированная верстка
source/ ........... Папка с исходниками
	jade/ ......... Jade шаблоны (препроцессор HTML)
	styl/ ......... Stylus шаблоны (препроцессор CSS)


//	ВОЗМОЖНЫЕ ОШИБКИ:
// ==================================================
- При открытии gulp-run.cmd консоль открывается и закрывается.
	Решение: запустить консоль вручную, ввести команду и прочитать ошибку

- Могут быть проблемы при использовании "[]" в названии папок в пути к проекту.
